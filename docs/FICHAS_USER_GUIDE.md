# Guía de Usuario - Fichas Técnicas

El módulo de Fichas Técnicas de CulinaryOS te permite estandarizar tus recetas, controlar costos y analizar la rentabilidad de tu carta.

## Funcionalidades Principales

### 1. Dashboard
Accede a una vista general de tus recetas.
- **KPIs**: Visualiza métricas clave como costo promedio y margen.
- **Búsqueda Inteligente**: Encuentra recetas por nombre o ingrediente.
- **Filtros**: Clasifica por categoría, dificultad, costo o margen.

### 2. Creación de Fichas
Para crear una nueva ficha:
1. Navega a **Nueva Ficha**.
2. Completa la información básica (nombre, porciones, categoría).
3. Añade ingredientes desde tu inventario. El costo se calcula automáticamente.
4. Define los pasos de preparación.
5. Establece un precio de venta sugerido o un margen objetivo.

### 3. Análisis de Rentabilidad
Esta herramienta te ayuda a tomar decisiones de negocio.
- **Gráfico de Rentabilidad**: Visualiza qué platos son "vacas lecheras" (alto margen, bajo costo) y cuáles necesitan optimización.
- **Simulador**: ¿Qué pasa si sube el precio del salmón? Usa el simulador para ver el impacto en tus márgenes antes de cambiar la carta.
- **Comparador**: Selecciona hasta 3 platos para comparar sus costos y márgenes lado a lado.

### 4. Importación de Recetas
Si ya tienes recetas en el sistema, puedes convertirlas a Fichas Técnicas con un solo clic.
1. Ve a "Importar Recetas" (si disponible en tu versión).
2. Selecciona las recetas que deseas convertir.
3. El sistema calculará los costos basados en el inventario actual.

## Consejos para Maximizar Ganancias
- Mantén actualizados los precios de tus ingredientes en el Inventario.
- Revisa el **Análisis de Rentabilidad** mensualmente.
- Apunta a un margen promedio del 70% para alimentos.
